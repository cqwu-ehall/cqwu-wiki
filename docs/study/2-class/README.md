# 第二课堂

## 简介

第二课堂学分是在第一课堂教学以外的时间，学生在教师的领导下所进行的全面提高综合素质的实践创新活动。

第二课堂学分主要与能否顺利拿到毕业证有关。

## 申报、查询方式

- [学生工作平台](http://ehall.cqwu.edu.cn/appShow?appId=6231227757845233)

团委服务 -> 第二课堂

## 计分细则

- [⬇️资源下载](/download/)

::: warning 注意

第二课堂学分需要主动在网上办事大厅中申报，经由班委、辅导员、学院批准之后即可加分。

第二课堂学分必须达到十学分才能颁发毕业证书。

:::
